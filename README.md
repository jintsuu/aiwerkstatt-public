# AI-Werkstatt Final Hand In

In diesem Branch ist der gesamte Code, der für die Findings bzw. Grafiken im Report genutzt wurde, gegliedert in Subfolder der jeweiligen Source-Repos.

## Projektstruktur

| Ordner | Beschreibung |
|--------|--------------|
| `flo/` | Code für die Datensätze **Loan** und **Nursery** |
| `patrick/` | Code für die Datensätze **Cardiovascular Health** (gen. "Cardio") und **California Housing** |

## Hinweis

> ⚠️ Es gab ein paar Probleme mit Git, weshalb es sein könnte, dass nicht alle Dateien die finalen Versionen sind, die im Report genutzt wurden. Sie sollten aber nicht allzu sehr von den Versionen im Report abweichen.
